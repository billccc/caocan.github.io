LeetCode | 553.最佳括号分配

发表于 2017-11-05 | 分类于 leetcode

题目描述：
https://leetcode.com/problems/optimal-division/description/

方法一：

这个方法比较暴力一些，就是利用递归来做。

  1. 当数组只有一个数字的时候，就直接把它返回即可
  2. 当数组中有两个以上的数字的时候，可以将数组分成左右两部分，那么我们要求的最大值一定就是(左边部分的最大值)/(右边部分的最小值)。所以可以创建一个类T来存储我们的最大值和最小值以及它们对应的字符串结果。
  3. 注意在返回构造字符串的时候要注意指针是不是已经走到数组末尾前的一个位置，如果已经到了那个位置，右边部分只有一个数，就不需要给它加括号了。

LeetCode | 669.裁剪二叉搜索树

发表于 2017-10-22 | 分类于 leetcode

原题地址：https://leetcode.com/problems/trim-a-binary-search-tree/description/

思路：
第一种是循环+递归方法做裁剪：

1.如果root是null，就直接返回null；
2.如果root值位于L和R之间，那么要求解的左子树就是将根的左子树带入递归算法得出的裁剪节点，求解的右子树就是根的右子树带入递归算法得出的裁剪节点；
3.如果root值不位于L和R之间，那么就让根一直查找到位于L和R的区间，再重复2的求解过程。
