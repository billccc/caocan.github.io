第十二节.stash命令

SourceTree：图形化的git操作工具

git stash：可以把当前未commit的工作放在存储区，该存储区是一个栈，最顶上是最新的stash
列出所有存储：git stash list
恢复最近存储：git stash pop
你指定版本号为stash@{1}的工作取出来：git stash apply stash@{1}
将所有的栈都应用回来：git stash clear
